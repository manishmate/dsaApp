| Pattern          | Approach                             | How to Identify                  |
| ---------------- | ------------------------------------ | -------------------------------- |
| Two Pointers     | Start from ends / same direction     | Sorted array, pair/triplet       |
| Sliding Window   | Expand + shrink window               | Subarray / substring + condition |
| Prefix Sum       | Precompute cumulative sum            | Sum queries / subarray sum       |
| Hashing          | Store frequency / seen elements      | Lookup, duplicates, fast search  |
| Kadane           | Local vs global max                  | Max/min subarray                 |
| Binary Search    | Divide search space                  | Sorted / rotated / answer space  |
| Monotonic Stack  | Maintain increasing/decreasing stack | Next greater/smaller             |
| Sorting          | Sort then process                    | Order matters                    |
| Greedy           | Local optimal choice                 | Min/max steps, reachability      |
| Interval         | Merge overlapping ranges             | Start/end ranges                 |
| Cyclic Sort      | Place at correct index               | Values in range [1..n]           |
| Matrix           | Row/column traversal                 | 2D grid problems                 |
| In-place         | Modify without extra space           | Space constraint                 |
| Divide & Conquer | Split + merge                        | Recursive breakdown              |
| Bit Manipulation | XOR / bit tricks                     | Missing/unique elements          |
